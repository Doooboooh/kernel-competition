📋 比赛规则

📝 参赛要求

  • 参赛者需完成实名认证
  • 支持个人或团队参赛（团队最多3人）
  • 每位选手只能参加一支队伍
  • 参赛者需同意并遵守比赛规则

💻 提交规范

  • 代码需通过 Git 提交到指定仓库
  • 每天最多提交 10 次
  • 提交后系统自动评测，实时更新分数
  • 以最高分数作为最终成绩

⚖️ 评分标准

  • 正确性 (60%): 测试用例通过率
  • 性能 (30%): 运行时间和内存占用
  • 代码质量 (10%): 可读性和规范性
  • 同分情况按提交时间排序

🚫 禁止行为

  • 禁止抄袭他人代码或使用非法手段
  • 禁止恶意攻击评测系统
  • 禁止多账号参赛
  • 违规者将取消参赛资格

❓ 常见问题

如何报名参赛？ ▼
点击页面顶部的"立即报名"按钮，完成账号注册和实名认证后即可参赛。团队参赛需由队长创建团队并邀请队员加入。
支持哪些编程语言？ ▼
目前支持 C/C++、Python、Java、Go、Rust 等主流编程语言。具体支持的语言版本请参考题目说明。
如何提交代码？ ▼
代码需要通过 Git 提交到我们提供的代码仓库。每次 push 后系统会自动触发评测，评测结果会在几分钟内更新到排行榜。
评测系统的运行环境是什么？ ▼
评测系统运行在 Linux 环境下，配置为 8核 CPU、32GB 内存。时间限制和内存限制会在每道题目中具体说明。
如何联系组委会？ ▼
可以通过邮件 contest@example.com 联系我们，或在官方 Discord 频道提问。工作日 24 小时内会回复。
